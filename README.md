# Política de Privacidade

Última atualização: 11 de fevereiro de 2026

Esta Política de Privacidade descreve como as informações são coletadas, utilizadas e protegidas pelo Bot de WhatsApp operado por Matheus Andrade de Souza, desenvolvido utilizando a plataforma n8n e a API oficial do WhatsApp (Meta Platforms, Inc.).

## 1. Informações Coletadas

Ao interagir com nosso bot via WhatsApp, podemos coletar as seguintes informações:

- Número de telefone
- Nome de perfil do WhatsApp
- Conteúdo das mensagens enviadas
- Data e horário das interações

## 2. Finalidade do Uso

As informações coletadas são utilizadas exclusivamente para:

- Responder às mensagens enviadas pelo usuário
- Fornecer atendimento automatizado
- Processar solicitações feitas pelo usuário
- Melhorar a experiência e funcionamento do serviço

## 3. Compartilhamento de Dados

Não vendemos, alugamos ou compartilhamos dados pessoais com terceiros.

Os dados podem ser processados por:

- Meta Platforms, Inc. (WhatsApp Cloud API)
- Serviços de hospedagem utilizados para execução do sistema

O compartilhamento ocorre apenas quando necessário para o funcionamento do serviço ou quando exigido por lei.

## 4. Armazenamento e Retenção

As mensagens podem ser armazenadas temporariamente para:

- Registro de interações
- Monitoramento de funcionamento
- Auditoria e segurança

Os dados são mantidos apenas pelo tempo necessário para cumprir suas finalidades.

## 5. Segurança

Adotamos medidas técnicas e organizacionais razoáveis para proteger as informações contra acesso não autorizado, alteração, divulgação ou destruição.

## 6. Direitos do Usuário

O usuário pode solicitar:

- A exclusão de seus dados
- Informações sobre dados armazenados
- Esclarecimentos sobre o uso das informações

## 7. Alterações nesta Política

Esta Política de Privacidade pode ser atualizada periodicamente. A versão mais recente estará sempre disponível nesta página.
